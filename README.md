# Pirkanmaan Betonilattiat

Landing page for a concrete flooring company in Pirkanmaa, Finland.

## Tech Stack

- Pure HTML/CSS/JS (Static Site)
- No build step required
- Optimized for Vercel deployment

## Project Structure

```
/
├── index.html      # Main landing page
├── media/          # Images folder
│   └── hero-concrete.jpg  # Hero background (add your image here)
├── vercel.json     # Vercel configuration
└── README.md
```

## Deploy to Vercel

### Option 1: Via Vercel Dashboard
1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com)
3. Import the repository
4. Click "Deploy" - no configuration needed!

### Option 2: Via Vercel CLI
```bash
npm i -g vercel
vercel
```

## Adding Hero Image

Place your hero image at `/media/hero-concrete.jpg` for the background to appear.

Recommended specs:
- Resolution: 1920x1080 or higher
- Format: JPG (optimized for web)
- File size: < 500KB
